Parent Element

Child element without any overrides. It inherits red color from the parent.

Child element with color: unset. Resets the color to inherit from the parent (red).

Child element with an inline color: blue, but color: unset in CSS. Resets to parent color (red).

Child element with an inline margin: 50px, but margin: unset in CSS. Resets to browser default (0px).

Child element without unset. Keeps its inline margin of 50px.
